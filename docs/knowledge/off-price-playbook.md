# Off-Price Playbook — DDS / Ross / Burlington / TJX 客户开发指南

> 业务方法论 + 月度推款日历 + 成本映射 + 行动清单。
> 适用对象：业务员、跟单、CEO 决策。
> 不适用对象：高端品牌 / Boutique 客户（逻辑完全不同）。

---

## 一、底层逻辑（30 秒摘要）

Off-Price 客户 ≠ 普通品牌零售。

| 维度 | Off-Price | 普通品牌 |
|------|-----------|---------|
| 核心诉求 | 快周转 + 低零售价 + 高性价比 | 品牌调性 + 设计差异化 |
| 客户买的是 | "可快速卖掉的货" | "可建立品牌印象的款" |
| Timing 容错 | 极低（推晚 1 个月即错过窗口） | 中等 |
| 成本敏感度 | 极高（FOB 误差 0.5 美金可能失单） | 中等 |
| Buyer 提前看货时间 | 4-8 个月 | 6-12 个月 |
| 决定下单的因素 | retail mapping + delivery + margin | trend + brand fit + creative |

**核心公式**：

> **Off-Price 成功 = Timing × 成本 × Delivery × Retail Mapping**
>
> 设计排第 5 位。

## 二、客户类型识别决策树（业务员必看）

收到客户询盘 / 拜访新客户时，先判断客户体系，决定推款策略。

```
客户名是否包含以下关键词？
├─ DDS / Ross / Burlington / TJX / Marshalls / HomeGoods → Off-Price 体系
│   策略：按本 Playbook 执行
│
├─ Lululemon / Athleta / Alo / Vuori → Premium Activewear
│   策略：本 Playbook 不适用，按品牌客户处理
│
├─ Target / Walmart / Costco → Mass Retail
│   策略：节奏类似 Off-Price 但更严格（compliance、social audit、test order）
│
├─ Boutique / DTC / Amazon brand → Brand / DTC
│   策略：小批量 + 高 trend + 高弹性，本 Playbook 不适用
│
└─ 不确定 → 问以下 3 个问题：
    1. 你们零售价主要区间是？  (≤30 美金 = Off-Price 倾向)
    2. 一个 PO 平均多少件？     (≥5000 件 = Off-Price / Mass)
    3. 提前多久看春夏货？       (4-8 月 = Off-Price)
```

## 四、重点月份拆解

### 1-3 月 ｜ 秋冬开发黄金季（全年最重要）

**客户视角**：准备 8-12 月销售，评估 fleece / 套装 / 保暖类。开始压 FOB 成本。

**重点推款**：

| 品类 | 女装 | 男装 | Activewear |
|------|-----|-----|------------|
| 套装 | fleece set / hoodie set / jogger set | fleece hoodie / jogger | fleece flare pants |
| 单品 | sherpa jacket / brushed leggings | thermal tops | cold-weather leggings / long sleeve top |
| 居家 | rib lounge set | — | brushed yoga pants |

**业务必做**：

- [ ] 准备秋冬 color card（含 holiday color 备选）
- [ ] 准备 cost-down 版本（同款不同布料 / 工艺简化版）
- [ ] 准备 packaging / carton optimization 方案
- [ ] 提前 fabric booking（避免 3 月底面料缺货）
- [ ] 算 retail mapping（用本文档第 5 节的成本表反推）

### 4-6 月 ｜ 秋冬补单 + 秋季 Activewear

**客户视角**：部分秋冬 PO 已下，开始做补充款 + fall activewear，开始看 BTS（Back to School）。

**核心关键词**：**Transition** — 能从夏卖到秋的款。
- 中等克重
- 可 layering
- 颜色偏中性
- 适合 school season

**重点推款**：crossover flare / basic legging / tennis skirt / bra set / yoga set / seamless set / lightweight fleece / modal tops。

### 7-8 月 ｜ Holiday + 冬季补货关键期

**客户视角**：圣诞 + gifting + 冬季补货 + 快反 + last-minute opportunity。

**Holiday 配色**：burgundy / forest green / chocolate / cream / red / black。

**重点推款**：fleece blanket hoodie / cozy lounge set / sherpa / gifting socks / pajama set / plush items。

**策略**：**Quick Turn 快反** — 此时客户已经来不及重新开发，现货布 + 现成版型 + 快速翻单 = 极强优势。

### 9-11 月 ｜ 春夏开发黄金期

**客户视角**：虽然美国入秋，buyer 已开始想明年春夏 / 准备 swim / shorts / 评估新颜色 / 评估运动趋势。

**Spring/Summer 配色**：pastel / sky blue / butter yellow / sage green / blush pink / light grey。

**重点推款**：shorts / tennis dress / tank tops / lightweight leggings / biker shorts / sports bra / running shorts / golf wear。

**核心策略**：**Trend + Cost** — 既要 trend 感，又要低成本。
- 做 trend simplification（取趋势的形，不要趋势的辅料）
- 用低成本实现 trend feeling
- 提供 retail-friendly version

### 12 月 ｜ 清仓 + 快速补货月

**客户视角**：年底库存压力 + 预算结算 + 清仓 + 快速补货 + 小单。

**核心策略**：**不要推太复杂产品**。客户此时更看重速度、价格、quick reorder、稳定性。

**重点推款**：black leggings / solid bra / fleece basics / white tops / seamless basics。

## 五、Retail → FOB 成本映射（按品类拆分）

原始单一表格容易误导。**按品类拆分**才能精准报价。

### Activewear 类（毛利结构相似）

| Retail USD | Basic Legging | Crossover Flare | Fleece Set | Yoga Set |
|------------|--------------|-----------------|------------|----------|
| 14.99 | 3.50-4.50 | 4.00-5.00 | — | 4.00-5.00 |
| 19.99 | 5.00-6.50 | 5.50-7.00 | 6.50-8.00 | 5.50-7.00 |
| 24.99 | 6.50-8.00 | 7.00-8.50 | 8.00-10.00 | 7.00-9.00 |
| 29.99 | 8.00-10.00 | 8.50-10.50 | 10.00-12.50 | 9.00-11.00 |

### Loungewear / Cozy 类

| Retail USD | Rib Lounge Set | Sherpa Jacket | Pajama Set |
|------------|---------------|---------------|------------|
| 19.99 | 6.00-7.50 | 7.50-9.00 | 5.50-7.00 |
| 24.99 | 7.50-9.00 | 9.50-11.50 | 7.00-8.50 |
| 29.99 | 9.50-11.00 | 11.50-14.00 | 8.50-10.50 |

### Bra / Small Item 类

| Retail USD | Sports Bra | Bra Set | Seamless |
|------------|-----------|---------|----------|
| 9.99 | 2.20-3.00 | — | 2.50-3.20 |
| 14.99 | 3.50-4.50 | 4.00-5.50 | 4.00-5.00 |
| 19.99 | 5.00-6.00 | 5.50-7.00 | 5.50-7.00 |

> 注：FOB 区间为参考值，需根据数量级、面料指定、装箱方式、Lab dip 数量等因素调整。

### 业务员必须会的反推路径

```
零售价 24.99 USD
  ÷ 客户毛利率 (~40%)      → 客户进货价 ~ 15.00 USD (Landed)
  − 海运 + duty + 装箱     → FOB 需 ~ 8-10 USD
  − 工厂利润 + 风险准备     → 厂家成本必须 < 6-7 USD
```

报价偏离这个反推 ≥ 15% → 几乎不可能拿到 PO。

## 六、Off-Price 适合 / 不适合品类

### ✅ 最容易做量

| 大类 | 具体品类 |
|------|---------|
| 女装 Activewear | flare legging / crossover flare / basic legging / yoga pant / fleece set / jogger set / lounge set |
| 家居 / 舒适 | pajama / cozy set / rib lounge / soft touch items |
| 低风险 evergreen | black legging / black flare / neutral bra / fleece hoodie |

### ❌ 不建议重点做

| 大类 | 原因 |
|------|------|
| 高复杂 fashion | heavy embellishment / couture / 难裁难缝 / 贵辅料 → 成本结构对不上 |
| 高单价 trend | 单价 ≥ 40 美金的 trend 款 → DDS / Ross 渠道不消化 |
| 极小众款 | 量上不来，无法摊薄开发成本 |

## 七、Monthly Action Checklist（业务员动作清单）

> 把"该做什么"从散落各处提取到一张表，避免漏动作。
> 业务员每月 1 号过一遍。

### 每月通用（不论季节）

- [ ] 检查 ETD / 出厂日逼近的订单是否有风险
- [ ] 拉本月对应 SKU 类别报价表（参考第 5 节）
- [ ] 给老客户发当月推款 lookbook
- [ ] 查看 Off-Price 客户当月该推的品类（参考第 3 节）

### 1 月 / 7 月 — 开发季启动

- [ ] 准备秋冬（1 月）/ 春夏（7 月）color card
- [ ] 准备 cost-down 版本
- [ ] 锁定面料 booking
- [ ] 算 retail mapping → FOB target 表

### 3 月 / 9 月 — 集中确认月

- [ ] 跟进所有报价转 PO 进度
- [ ] 工厂产能提前 sync（避免 4 月 / 10 月集中下单时排不进去）
- [ ] 装箱 / 唛头标准提前确认

### 6 月 / 12 月 — 补货 + 库存关键期

- [ ] 现货布 / 现货版型清单更新
- [ ] 快反报价表准备好（提前 3 天回复 = 拿单关键）
- [ ] 主动询问客户清仓 / 补货机会

## 八、公司内部应建立的系统化资产

### 1. Off-Price Cost Library（成本库）

记录：FOB target / retail mapping / 包装 / shipping / duty / margin（按品类、按数量级、按面料）。

### 2. Off-Price Fit Library（版型库）

记录：哪种 fit 最易卖 / 哪种 rise 最被接受 / 哪种 inseam 最稳定 / 哪种 waistband 最少投诉。

### 3. Seasonal Recommendation Calendar（月度推款日历）

业务员每月必须知道：现在该推什么、该问什么、该备什么。
→ 本文档第 3 节是 v1 版本，落到系统后由 `lib/agent/industryKnowledge.ts` 程序化提供。

## 九、客户真正在意的（而非 buyer 嘴上说的）

| Buyer 嘴上说 | 真正在意 |
|-------------|---------|
| "设计要好看" | 能不能卖 |
| "我们要 trend" | 有没有 margin |
| "看看新款" | 能不能稳定补货 |
| "再考虑下" | delivery 稳不稳定 |
| "客户反馈不错" | return rate 高不高 |
| "下次再合作" | fit 是否稳定 |

---

## 十、一句话总结

> 美国 Off-Price 本质不是"便宜货"。
> 而是一套极其成熟的：**低价、高周转、大规模、强 timing 的供应链操作系统。**
>
> 想长期做下来，靠的不是"最会设计"，而是"最懂 timing + 成本 + delivery + retail mapping"。
