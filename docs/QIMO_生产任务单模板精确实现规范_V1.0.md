# QIMO 生产任务单标准模板 V1.0

## 设计原则

本文件不是“参考风格”，而是生产任务单的**母版**。系统导出时必须直接复制母版工作表及其样式，再按映射写入数据。

禁止根据截图重新画表；禁止使用近似颜色、近似列宽、近似合并规则；禁止用 HTML/CSS 另做一套外观。

模板来源：`QIMO_生产任务单标准模板_V1.0.xlsx`

## 系统实现规则

1. 将母版放入仓库固定路径，例如：  
   `public/templates/QIMO_生产任务单标准模板_V1.0.xlsx`

2. 导出时必须：
   - 加载母版；
   - 复制目标 Sheet；
   - 仅向指定数据单元格写值；
   - 不重建样式；
   - 不调用 autofit；
   - 不改变行高、列宽、合并、打印设置、字体、颜色、边框、对齐、换行。

3. 动态颜色或尺码超过母版容量时：
   - 不得挤压或缩小原版；
   - 应复制款色明细行或新增附页；
   - 新增行必须从对应模板行完整复制样式及合并规则；
   - 主表结构保持不变。

4. 动态尺寸表：
   - 默认继承第二个 Sheet；
   - 尺码列超出 S/M/L 时，按模板列组完整复制，不使用自动列宽；
   - 表头、边框、字体及单位必须保持一致。

5. 文本写入：
   - 保留模板中的固定描述和指定措辞；
   - 系统数据只替换占位数据；
   - 固定警示语、表头、字段名称不得改写；
   - 关键风险词红色粗体格式必须保留。

6. 验收方式：
   - 导出文件与母版逐项比较：Sheet 名、used range、合并区域、列宽、行高、字体、字号、颜色、填充、边框、对齐、换行、打印方向、纸张和缩放；
   - 任一不同即失败；
   - 增加自动化 workbook-style regression test。

## 推荐字段映射

主表至少映射：

- 订单号
- 客户名称
- 下单日期
- 品名
- 原料/成分
- 交期
- 面料克重
- 总数量
- 款号
- 主布颜色
- 箱数
- 各尺码数量
- 客户包装
- 面料名称
- 每套/每件单耗及单位
- 产前样
- 船样
- 成衣辅料
- 包装辅料
- 裁剪要求
- 缝制要求
- 检验要求
- 包装要求
- 装箱要求
- 注意事项
- 签收人/签收时间

尺寸表至少映射：

- 款号/标题
- 序号
- 测量部位
- 尺码列
- 标准尺寸
- 上衣与下装两组测量数据

## 不得遗漏的测试

- 模板文件缺失时 fail closed，不能悄悄生成旧版式。
- 中文 Sheet 名和中文固定描述保持不变。
- 合并区域完全一致。
- 行列尺寸完全一致。
- 打印设置完全一致。
- 颜色与字体完全一致。
- 同一份订单重复导出样式 hash 一致。
- 只有业务数据单元格允许变化。
