# 生产中心 G–K Preview 验收

仅使用一次性人工测试订单。禁止修改真实订单 1022961，禁止点击最终采购或生产提交。

## 1. 生产延期审批

- [ ] 用生产跟单提交一个“不改变客户承诺交期”的生产延期。
- [ ] 生产主管打开订单 → 延期申请 → 点击“确认这一步”。
- [ ] 预期：主管可批准/驳回；申请人自己看不到可执行按钮；截图审批链和结果。
- [ ] 再建立一个“改变客户承诺交期”的测试延期。
- [ ] 预期：生产主管确认后转业务经理；生产主管不能执行业务经理步骤；截图当前责任角色。

## 2. 排产内部单号

- [ ] 打开生产中心 → 排产工作台。
- [ ] 在搜索框输入人工测试订单的内部单号。
- [ ] 预期：卡片显示不同的 QM 单号、内部单号、客户 PO、款号；缺值显示 `—`。

## 3. 生产接单与跟单指派

- [ ] 生产主管在“待生产接单/待分配生产跟单”找到测试订单。
- [ ] 点击订单并选择一名在职、角色为“生产”的跟单。
- [ ] 预期：固定主管、业务、财务、QC 节点不被改派；重复选择同一人不产生重复节点。
- [ ] 由被指派人员登录。
- [ ] 预期：能立即看到该订单；截图双方工作台。

## 4. 单耗小数和单位

- [ ] 在人工订单报价基线分别准备 `1.05 米/件`、`0.75 平方米/件`、`0.032 kg/件` 示例。
- [ ] 打开生产启动/开裁 → 上线工艺确认报告。
- [ ] 预期：报价值和单位自动继承，输入、保存、刷新后小数完全不变。
- [ ] 将实际单位改为与报价不同。
- [ ] 预期：系统明确提示单位不一致且不进行静默换算。

## 5. 工厂完整性

- [ ] 打开排产工作台并展开派工。
- [ ] 用名称和工厂代码搜索此前缺失的在合作工厂。
- [ ] 预期：所有 active/trial 且未删除的生产工厂均可见；推荐排在前面，能力或产能不匹配显示原因但不隐藏。
- [ ] 预期：同一工厂只显示一次，已选工厂仍有效。

## 6. 生产主管队列

- [ ] 验证：待生产接单、待分配生产跟单、待选工厂、延期待审批、异常待处理、今日应开裁、今日应上线、已超期。
- [ ] 每张卡应显示订单标识、进入原因、下一步和“去处理”链接。
